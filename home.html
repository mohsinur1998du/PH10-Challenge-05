What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page in a tree-like format, where each part of the document (HTML elements, attributes, text) is represented as a node. This model allows scripts (like JavaScript) to dynamically access, modify, add, or delete elements in a web page.

How do you select an element from DOM?

You can select an element from the DOM using various JavaScript methods. These methods allow you to target elements based on their ID, class, tag, or other attributes.

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique in JavaScript for handling events efficiently when you have multiple elements that need similar event handling. Instead of attaching event listeners to each individual element, you attach a single event listener to a parent element.

How do you manipulate an element's attributes and styles using DOM?

To manipulate an element's attributes and styles using the Document Object Model (DOM) in JavaScript, you can use various methods and properties, like Manipulating attributes, Manipulating styles, Changing content.
